# Caio UI 
[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)

Exemplo de Design System usando o [stitches](https://stitches.dev) e [radix](https://www.radix-ui.com).

## Instalação

No seu terminal instale:

```bash
# With npm
npm install @caio-ui/react
```

## Uso

No seu projeto react, no arquivo `App.jsx` ou `App.tsx`, faça:

```javascript
import React from 'react'
import { Button } from '@caio-iu/react'

export function App(){
    return(
        <Button variant='primary'>Exemplo de botão</Button>
    )
}

```
