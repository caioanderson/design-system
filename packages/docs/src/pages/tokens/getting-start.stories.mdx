# Caio UI 

## Introdução

Nessa etapa de instalação vão está disponiveis :
- [ ] Cores.
- [ ] Fontes (tamanho, peso, tipo).
- [ ] Tamanho da linha.
- [ ] Espaçamentos.
- [ ] Valores de borda.

## Instalação

No seu terminal instale:

```bash
# With npm
npm install @caio-ui/tokens
```

## Uso

No seu projeto react, no arquivo `App.jsx` ou `App.tsx`, faça:

```javascript
import React from 'react'
import { colors, fontSizes, fontWeights, fonts } from '@caio-iu/tokens'

const styles = {
   color: colors.gray800,
   fontSize: fontSizes.md,
   fontFamily: fonts.default,
   fontWeights: fontWeights.regular
}

export function App(){
    return(
        <span style={styles}>
            Lorem ipsum dolor sit amet, consectetur adipiscing elit.
         </span>
    )
}

```
